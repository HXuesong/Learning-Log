# matlab
```matlab
x1 = -5 : 0.1 : 5;
y1 = x .^ 2;
plot(x1, y1);

x2 = -5 : 0.1 : 5;
y2 = x .^ 3;
plot(x2, y2);

%matlab在使用plot()函数时，都会把上一次的图形给覆盖掉
%最后只会打印一条三次曲线
```



- hold on
```matlab
x1 = -5 : 0.1 : 5;
y1 = x .^ 2;
plot(x1, y1);

hold on;

x2 = -5 : 0.1 : 5;
y2 = x .^ 3;
plot(x2, y2);

%hold on可以使两张图像绘制在同一个版面上
%将二次曲线、三次曲线打印在同一个版面上
```



- grid on
```matlab
x1 = -5 : 0.1 : 5;
y1 = x .^ 2;
plot(x1, y1);

hold on;

x2 = -5 : 0.1 : 5;
y2 = x .^ 3;
plot(x2, y2);

grid on;

%grid on可在两张图像背景上显示网格
```



- title()
```matlab
x1 = -5 : 0.1 : 5;
y1 = x .^ 2;
plot(x1, y1);

hold on;

x2 = -5 : 0.1 : 5;
y2 = x .^ 3;
plot(x2, y2);

grid on;
title('x^2 VS x^3');

%title()可为图像上部添加小标题
```



- xlabel()、ylabel()
```matlab
x1 = -5 : 0.1 : 5;
y1 = x .^ 2;
plot(x1, y1);

hold on;

x2 = -5 : 0.1 : 5;
y2 = x .^ 3;
plot(x2, y2);

grid on;
title('x^2 VS x^3');
xlabel('x-axis');
ylabel('y-axis');

%xlabel()可为图像x轴添加小标题
%ylabel()可为图像y轴添加小标题
```



- subplot()
```matlab
x = -4 : 0.1 :4;
y1 = sin(x);
y2 = sin(2 .* x);
y3 = sin(3 .* x);
y4 = sin(4 .* x);

subplot(2, 2, 1);
plot(x, y1);
title('y = sin(x)');

subplot(2, 2, 2);
plot(x, y2);
title('y = sin(2x)');

subplot(2, 2, 3);
plot(x, y3);
title('y = sin(3x)');

subplot(2, 2, 4);
plot(x, y4);
title('y = sin(4x)');

%subplot(a, b, c)
%a表示共几行
%b表示共几列
%c表示定位到第几个图像的位置
%绘制2x2显示的四张图
```
```matlab
x = -4 : 0.1 :4;
y1 = cos(x);
y2 = cos(2 .* x);
y3 = cos(4 .* x);

subplot(2, 2, 1);
plot(x, y1);
title('y = cos(x)');

subplot(2, 2, 2);
plot(x, y2);
title('y = cos(2x)');

subplot(2, 2, [3, 4]);
plot(x, y3);
title('y = cos(3x)');

%subplot(2, 2, [3, 4]);
%[3, 4]表示第三张图占据[3, 4]即下部空间
```



- meshgrid()
- surf()
```matlab
x = -3 : 0.1 : 3; 
y = -3 : 0.1 : 3;

[X, Y] = meshgrid(x, y);

Z = X .^2 + Y .^ 2;

surf(X, Y, Z);

%[X, Y] = meshgrid(x, y)可得到一组[X, Y]
%X和Y为矩阵，可形成平面上所有点的坐标
%surf()用于三维曲面的绘制
```
